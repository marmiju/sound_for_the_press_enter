A Visual Studio Code extension that plays a sound when you press the "Enter" key.

## Features

*   Plays a sound every time you press the "Enter" key.
*   Easy to enable and disable.

## Installation

1.  Launch Visual Studio Code.
2.  Go to the Extensions view (Ctrl+Shift+X).
3.  Search for "Enter Sound".
4.  Click "Install" to install it.
5.  Restart Visual Studio Code.

## Usage

Once the extension is installed, it will be automatically enabled. Every time you press the "Enter" key, you will hear a sound.

To disable the extension, go to the Extensions view, find "Enter Sound", and click "Disable".
